Tutorial

Before getting creative, here are some resources that will be used in this tutorial

  • Vector Graphics software (like Adobe Illustrator)
  • Inkscape (free vector graphics software with SVG font editor)
  • Iconfont canvas template by Heydon Pickering
  • Font Converter (like freefontconverter.com)
  • Web Font Converter / @font-face Generator(like fontsquirrel.com)

Create Icon Font

When creating icons for a font they have to be in svg (scalable vector graphic) format in order to be free scalable.

  1. open the Iconfont canvas template in a vector graphic software and create an icon

  2. save icon as icon.svg

  3. open the open the Iconfont canvas template in Inkscape

  4. setting up the workspace Object -> Fill and Stroke Object -> Align and Distribute Text -> SVG Font Editor

  5. import the icon.svg and center it on the canvas

  6. to make the icon work for a font it needs to be converted to a path shape Path -> Object to Path

  7. to make a glyph from the shape

    • (1) create a new glyph
    • (2) enter a matching string
    • (3) enter the matching string into the preview text
    • (4) and select Get curves from selection

    troubleshooting

    if the following steps with get curves from selection don't work apply the following steps aswell:

    1. Path -> Combine
    2. Path -> Union
    3. Fill -> Solid Color

  8. save icon as iconset.svg

  9. convert the svg to a font with a service like freefontconverter.com
