The CSS property for leading is letter-spacing

Syntax

letter-spacing:  normal | length
normal

default value
Normal spacing for current font.

length

normal = 0
Fixed space between characters. Values can be also negative with common units [px, pt, em, rem, ...more info]

Kerning

So far there is no implemented font-kerning property (W3 draft)

possible workaround:

<span style="letter-spacing: {value}{unit}>
