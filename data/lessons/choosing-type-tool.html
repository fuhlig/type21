When designing a website there is often a discrepancy between mockups and coded websites. Using graphic and layout software like Photoshop or InDesign the designer has the right tools at hand to create the desired typographic style. To recreate this (if even possible...) needs a lot more effort with HTML & CSS, eventhough designing in the browser becomes more and more popular.

For better typography with less hassle typecast.com was created.

Typecast

Typecast is a web app to create beautiful crafted typography for the web.

Some of the core features include

  • WYSIWYG editor with preview
  • easy comparison of font combinations
  • 23,000+ web fonts
  • baseline grid
  • styleguide (with fonts, styles, colors, ...)
  • CSS export clean ready-to-use code
