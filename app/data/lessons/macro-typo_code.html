Leading

The CSS property for leading is line-height

Syntax

line-height: normal | number | length | percentage
normal

default value
Depending on user agent

number

Value without a unit to multiply the line height by the element's font size

length

Fixed line height with common units [px, pt, em, rem, ...more info]

percentage

Relative to the element's font size
