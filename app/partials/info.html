type21 is a project for the interactivity class at the LYIT promoting typography.

While creating this site and content I also learned new interesting facts about typography and tried to integrate it here (like font sizing, icon fonts, webfonts). So I hope you learn something new aswell.

Unfortunatly some features are not quite implemented or fully functional yet BUT I plan to keep working on it. Using this site as a digital playground to optimize and add features in terms of design and development like making this site fully responsive. With relative small effort it already works to some extend - check it out!

Any further development can be followed at the github repo
